# Bank Loan Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes bank loan data and provides insights into loan amount, customer credit score, annual income, loan purpose, home ownership, and loan terms.

## Dashboard Features
- Total Loan Amount
- Total Loan Count
- Average Credit Score
- Average Annual Income
- Loan Amount by Purpose
- Loan Amount by Home Ownership
- Loan Distribution by Term
- Monthly Debt vs Credit History
